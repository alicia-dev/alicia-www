Meet Alicia

Alicia is an open-source, lightweight and multi-platform game framework.

FEATURES

• Cross-platform


This doesn't mean written-once run-everywhere. However, you can port your game to any supported platform within few tweaks.

• Software Render


It might be a bit slow, but you will get same result regardless of device the game is running on.

• Independence


Alicia doesn't rely much on 3rd-party libraries. But you still need the standard and OS-specified ones.

• Open-source


It's free as in freedom. You may sell the game you made. You can even sell Alicia itself.

• Easy & Simple


It's extreme easy to get started. You don't have to learn through a very huge document of functions.

• Data-oriented


Forget about the objects that ruin performance. Let's make the games run fast.

FAQs

Please read through this FAQs before asking us any question. If you have a new question, feel free to contact us!

What is Alicia?

It's an open-source framework for game creation. You use it to create a video game.

Can I eat it now?

Pretty not soon. This project is very incomplete. It's still in early development stage.

What platform does Alicia run on?

You can build your game on Windows and some UNIX-like systems. Alicia engine is planned to support running the games on Windows, Linux, Mac, Android, iOS, Tizen, FreeBSD, Haiku.

How much does it cost to use Alicia?

Alicia is free as in freedom, and open-source. You can use it for both non-commerical and commerical project. No royalty needed.

When will I be able to use it?

Not soon. Please don't ask. This project is being worked on by a single amateur 16-years-old with a 12-years-old-like's brain.

Can I help you with something in this project?

Of course! I'm glad you're interested. Please help me fix the damn documention. PRs are also welcomed!
